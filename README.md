POC of Java based framework for E2E UI testing of http://demos.telerik.com/kendo-ui/websushi/ website.

This project is meant for educational purposes only. 

#### Dependencies 

    assertj: 3.6.1
    jbehave: 4.1
    junit: 4.11
    selenium-java: 3.0.1

#### System requirements 

* Java 8
* Gradle
* Chrome browser
